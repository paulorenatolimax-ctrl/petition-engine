## 🚀 COMO FUNCIONA

### Fluxo de 4 Prompts Sequenciais

```
PROMPT 1: MAPEAMENTO DE PERFIL
├─ Extrai informações do CV
├─ Organiza em 10 categorias
├─ Mapeia aos 10 critérios EB-1A
└─ Saída: Relatório de Mapeamento (2-3 horas)

        ↓

PROMPT 2: ANÁLISE DE CRITÉRIOS
├─ Analisa cada um dos 10 critérios
├─ Classifica força de evidências
├─ Identifica oportunidades de fortalecimento
└─ Saída: Análise Profunda (3-4 horas)

        ↓

PROMPT 3: CÓDIGOS OCUPACIONAIS
├─ Identifica 3 códigos BLS alternativos
├─ Valida requisitos educacionais
├─ Evita profissões com restrições
└─ Saída: Recomendação de Códigos (2-3 horas)

        ↓

PROMPT 4: RELATÓRIO ESTRATÉGICO
├─ Consolida toda a análise
├─ Estrutura roadmap de ação
├─ Define cronograma de implementação
└─ Saída: Roadmap Completo (2-3 horas)

TOTAL: ~10-13 horas por cliente
```

## 🎓 DIFERENÇAS CRÍTICAS: EB-1A vs EB-2 NIW

Este assistente é **100% EB-1A**, não EB-2 NIW:

| Aspecto | EB-1A | EB-2 NIW |
|--------|-------|---------|
| **Foco** | Habilidade extraordinária | Interesse nacional |
| **Critérios** | 10 critérios regulatórios | 3 prongs |
| **Proposed Endeavor** | Continuidade de trabalhos (flexível) | Negócio específico |
| **Benefício** | Implícito | Explícito |
| **Flexibilidade** | Alta | Média |

**Importante:** Não misture estratégias. Este é exclusivamente EB-1A.
