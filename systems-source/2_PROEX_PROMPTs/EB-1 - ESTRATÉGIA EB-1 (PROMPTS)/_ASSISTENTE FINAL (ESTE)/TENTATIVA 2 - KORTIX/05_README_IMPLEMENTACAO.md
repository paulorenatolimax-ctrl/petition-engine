## 🎯 DIFERENÇAS CRÍTICAS: EB-1A vs EB-2 NIW

Este assistente foi desenvolvido **especificamente para EB-1A**, não EB-2 NIW. As diferenças são críticas:

| Aspecto | EB-1A | EB-2 NIW |
|--------|-------|---------|
| **Foco** | Habilidade extraordinária | Interesse nacional |
| **Critérios** | 10 critérios regulatórios | 3 prongs |
| **Proposed Endeavor** | Continuidade de trabalhos (flexível) | Negócio específico |
| **Benefício** | Implícito (para o país) | Explícito (interesse nacional) |
| **Flexibilidade** | Alta (emprego, autônomo, etc.) | Média (deve ser negócio) |
| **Foco de Análise** | Excelência passada | Capacidade futura |

**Importante:** Não misture estratégias EB-1A com EB-2 NIW. Este assistente é exclusivamente EB-1A.

**NUNCA use:**

**ANTES de executar qualquer prompt:**

**NUNCA pule etapas:**
- ✅ Prompt 1 → Prompt 2 → Prompt 3 → Prompt 4
- ❌ Não execute Prompt 3 sem Prompt 2
- ❌ Não execute Prompt 4 sem Prompt 3
- ❌ Não execute múltiplos prompts simultaneamente

### 4. Qualidade de Saída

**SEMPRE verifique:**
- ✅ Linguagem profissional (sem emojis, sem chatbot)

## 🆘 TROUBLESHOOTING

### Problema: "Não tenho certeza qual código usar"

**Solução:**
1. Consulte `GUIA_CRITICO_CODIGOS_OCUPACIONAIS_EB1A.md`
2. Siga o protocolo de validação
3. Se ainda tiver dúvida, **PAUSE e consulte o consultor**
4. Nunca adivinhe

### Problema: "O RAG não está retornando informações"

**Solução:**
1. Verifique se o arquivo foi carregado
2. Tente reformular a pergunta
3. Consulte o arquivo PDF diretamente
4. Se persistir, recarregue o RAG

### Problema: "A saída não parece profissional"

**Solução:**
1. Revise a linguagem (remova emojis, chatbot)
2. Verifique fundamentação (cite RAGs)
3. Valide completude (todos os critérios?)
4. Revise com o consultor

### Problema: "Cliente quer pular etapas"

**Solução:**
1. Explique por que a sequência é importante
2. Mostre que cada prompt depende do anterior
3. Mantenha a sequência rigorosamente
4. Não faça exceções
