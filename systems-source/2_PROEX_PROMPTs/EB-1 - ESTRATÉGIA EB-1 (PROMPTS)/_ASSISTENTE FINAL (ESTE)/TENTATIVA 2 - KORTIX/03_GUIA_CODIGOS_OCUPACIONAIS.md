# 🚨 GUIA CRÍTICO: CÓDIGOS OCUPACIONAIS PARA EB-1A
## Protocolo de Segurança e Precisão - Prompt 3

**Versão:** 1.0 - Dezembro 2025  
**Criticidade:** MÁXIMA  
**Responsabilidade:** Consultor + Assistente

---

## ⚠️ AVISO CRÍTICO

**A seleção incorreta de códigos ocupacionais pode resultar em NEGAÇÃO da petição.**

Este guia estabelece protocolos invioláveis para garantir que os códigos selecionados sejam:
1. ✅ Precisos e alinhados com o perfil do cliente
2. ✅ Compatíveis com requisitos educacionais
3. ✅ Livres de conflitos com licenciamento profissional
4. ✅ Estrategicamente adequados para EB-1A

---

## 📋 PROTOCOLO DE SELEÇÃO DE CÓDIGOS

### PASSO 1: ANÁLISE DO PERFIL DO CLIENTE

Antes de qualquer busca no BLS, responda estas perguntas:

#### Formação Acadêmica
- [ ] Qual é o grau mais alto? (Bacharelado / Mestrado / Doutorado / Outro)
- [ ] Em qual área? (Engenharia / Administração / Ciências / Artes / Outro)
- [ ] De qual país? (Importante para validação de diplomas)
- [ ] Tem certificações profissionais? (Quais?)

#### Experiência Profissional
- [ ] Quais foram os cargos principais?
- [ ] Quais foram as responsabilidades concretas?
- [ ] Em quais setores trabalhou?
- [ ] Qual foi o nível de senioridade?

#### Atividades Específicas
- [ ] Que softwares/ferramentas domina?
- [ ] Que metodologias conhece?
- [ ] Que tipos de projetos liderou?
- [ ] Que tipo de análise/pesquisa realiza?

---

### PASSO 2: IDENTIFICAR PROFISSÕES COM RESTRIÇÕES

#### ⛔ PROFISSÕES QUE EXIGEM LICENCIAMENTO NOS EUA

Se o cliente trabalha em uma destas áreas, **NUNCA use o código direto**:

| Profissão | Código ERRADO ❌ | Código CORRETO ✅ | Razão |
|-----------|-----------------|------------------|-------|
| **Contador** | 13-2011 (Accountant) | 11-3031 (Financial Manager) | Exige CPA nos EUA |
| **Médico** | 29-1069 (Physician) | 11-9111 (Medical/Health Services Manager) | Exige licença médica |
| **Engenheiro** | 17-2011 (Engineer) | 11-9041 (Engineering Manager) | Exige PE (Professional Engineer) |
| **Advogado** | 23-1011 (Lawyer) | 11-3011 (Administrative Services Manager) | Exige bar exam |
| **Psicólogo** | 29-1066 (Psychologist) | 11-9111 (Health Services Manager) | Exige licença |
| **Arquiteto** | 17-1011 (Architect) | 11-9041 (Engineering Manager) | Exige licença |

**REGRA DE OURO:** Se a profissão exige licenciamento/validação de diploma nos EUA e o cliente não tem, use a versão "Manager" ou "Administrative" do código.

---

### PASSO 3: VERIFICAR REQUISITOS EDUCACIONAIS

Para CADA código que você considerar, verifique no BLS:

**Acesse:** https://www.bls.gov/ooh/[categoria]/[profissão].htm

**Procure pela seção:** "How to Become a [Profissão]"

**Identifique:**
- [ ] Educação mínima exigida
- [ ] Certificações necessárias
- [ ] Experiência mínima
- [ ] Treinamento específico

#### Matriz de Compatibilidade

| Requisito BLS | Cliente tem Bacharelado | Cliente tem Mestrado | Cliente tem Doutorado | Compatível? |
|---------------|------------------------|----------------------|----------------------|------------|
| Nenhum específico | ✅ | ✅ | ✅ | SIM |
| Bacharelado | ✅ | ✅ | ✅ | SIM |
| Mestrado | ❌ | ✅ | ✅ | SIM |
| Doutorado | ❌ | ❌ | ✅ | SIM |
| Licença específica | ❌ | ❌ | ❌ | NÃO (use alternativa) |

**REGRA:** Se o cliente não atende ao requisito educacional, o código é INCOMPATÍVEL.

---

### PASSO 4: BUSCAR NO BLS

#### Acesso Correto

1. Vá para: https://www.bls.gov/ooh/
2. Navegue pela categoria apropriada
3. Procure por ocupações que correspondam às atividades do cliente
4. **NÃO use a busca genérica** - navegue pelas categorias

#### Categorias Principais do BLS

- **Business and Financial Operations** (13-0000)
- **Computer and Mathematical** (15-0000)
- **Architecture and Engineering** (17-0000)
- **Life, Physical, and Social Science** (19-0000)
- **Management** (11-0000)
- **Professional and Related** (21-0000, 23-0000, 25-0000, 27-0000, 29-0000)

---

### PASSO 5: VALIDAR CADA CÓDIGO

Para cada código que você selecionar, preencha esta checklist:

#### Checklist de Validação

**Código:** [XX-XXXX]  
**Nome:** [Nome Oficial]  
**URL:** [URL Completa]

- [ ] URL é do BLS.gov (não de outro site)
- [ ] URL é específica para esta ocupação (não página genérica)

**Resultado:** ✅ APROVADO / ❌ REJEITADO

## 🎯 EXEMPLOS PRÁTICOS

### Exemplo 1: Cliente é Contador

**Perfil:**
- Formação: Bacharelado em Contabilidade
- Experiência: 10 anos como contador em empresa privada
- Atividades: Contabilidade financeira, auditoria, planejamento fiscal

**Análise:**
- ❌ NÃO usar: 13-2011 (Accountant) - exige CPA nos EUA
- ✅ USAR: 11-3031 (Financial Manager) - compatível com experiência, não exige licença

**Justificativa:**
"Embora o cliente tenha experiência como contador, a profissão de Accountant nos EUA exige certificação CPA. Portanto, recomenda-se o código 11-3031 (Financial Manager), que reflete as atividades de gestão financeira que o cliente realizará nos EUA, sem exigir licenciamento específico."

---

### Exemplo 2: Cliente é Engenheiro de Software

**Perfil:**
- Formação: Bacharelado em Engenharia de Software
- Experiência: 8 anos desenvolvendo software em startups
- Atividades: Desenvolvimento, arquitetura de sistemas, liderança técnica

**Análise:**
- ✅ USAR: 15-1252 (Software Developers) - compatível, não exige licença
- ✅ USAR: 15-1243 (Computer Network Architects) - se trabalhou com infraestrutura
- ✅ USAR: 11-9121 (Computer and Information Systems Managers) - se liderou equipes

**Justificativa:**
"O cliente tem experiência em desenvolvimento de software, que se alinha com o código 15-1252. Adicionalmente, se o cliente liderou equipes técnicas, o código 11-9121 também é apropriado. Nenhum destes códigos exige licenciamento específico."

---

### Exemplo 3: Cliente é Médico

**Perfil:**
- Formação: Doutorado em Medicina (MD)
- Experiência: 12 anos como médico clínico
- Atividades: Atendimento clínico, pesquisa, ensino

**Análise:**
- ❌ NÃO usar: 29-1069 (Physician) - exige licença médica nos EUA
- ✅ USAR: 11-9111 (Medical and Health Services Manager) - compatível com experiência
- ✅ USAR: 19-1042 (Medical Scientists) - se fez pesquisa significativa
- ✅ USAR: 25-1071 (Health Specialties Teachers) - se trabalhou em ensino

**Justificativa:**
"Embora o cliente seja médico, a prática de medicina nos EUA exige licença específica (MD/DO + residência + board certification). Portanto, recomenda-se o código 11-9111 (Medical and Health Services Manager), que reflete as atividades de gestão em saúde que o cliente pode realizar nos EUA. Se o cliente tem pesquisa significativa, 19-1042 também é apropriado."

---

### Exemplo 4: Cliente é Pesquisador em Biologia

**Perfil:**
- Formação: Doutorado em Biologia Molecular
- Experiência: 7 anos em pesquisa acadêmica
- Atividades: Pesquisa, publicações, orientação de alunos

**Análise:**
- ✅ USAR: 19-1029 (Biological Scientists) - compatível, não exige licença
- ✅ USAR: 25-1043 (Biological Science Teachers) - se trabalhou em ensino
- ✅ USAR: 11-9121 (Computer and Information Systems Managers) - se liderou laboratório

**Justificativa:**
"O cliente tem experiência em pesquisa biológica, que se alinha com o código 19-1029. Não há restrições de licenciamento. Se o cliente também trabalhou em ensino ou liderança de laboratório, os códigos 25-1043 ou 11-9121 também são apropriados."

## 🔍 VERIFICAÇÃO FINAL

Antes de apresentar os três códigos ao cliente, execute esta verificação final:

### Checklist Final

- [ ] Os três códigos são do BLS.gov

- [ ] Todos os códigos são compatíveis com a formação educacional do cliente
- [ ] Os códigos refletem as atividades concretas do cliente
- [ ] Os códigos são relevantes para o mercado de trabalho dos EUA
- [ ] Os códigos são estrategicamente adequados para EB-1A
- [ ] Não há conflitos entre os códigos
- [ ] A justificativa para cada código é clara e objetiva

**Se ALGUM item não estiver marcado, REVISE antes de apresentar.**

## 📞 QUANDO PEDIR AJUDA

Se você encontrar qualquer uma destas situações, **PAUSE e consulte o consultor responsável:**

1. ❓ Cliente trabalha em profissão que exige licenciamento e você não tem certeza qual código usar
2. ❓ Cliente tem formação de país diferente e você não tem certeza sobre equivalência
3. ❓ Cliente trabalha em área muito especializada e não encontra código exato no BLS
4. ❓ Cliente tem múltiplas carreiras e você não tem certeza qual código priorizar
5. ❓ Você encontra conflito entre requisitos educacionais e experiência do cliente

**Não adivinhe. Consulte.**
