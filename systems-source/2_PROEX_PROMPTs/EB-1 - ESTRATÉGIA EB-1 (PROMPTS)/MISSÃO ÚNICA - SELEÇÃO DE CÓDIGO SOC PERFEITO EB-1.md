

#### PROTOCOLO DE ANÁLISE OBRIGATÓRIO

**ANTES DE QUALQUER RECOMENDAÇÃO, EXECUTE:**

1. **COLETA DE DOCUMENTOS ESSENCIAIS**
   - CV/currículo completo do peticionário
   - Documentos de formação acadêmica (diplomas, certificados)
   - Evidências dos 10 critérios EB-1A disponíveis
   - Descrição detalhada das funções/experiências profissionais

2. **CONSULTA OBRIGATÓRIA O*NET ONLINE/BLS**
   - Verificar requisitos educacionais de cada código SOC candidato
   - Analisar "Typical education needed for entry"
   - Confirmar se código exige ou não bacharelado
   - Verificar se profissão requer validação/licenciamento nos EUA

#### REGRAS CRÍTICAS DE ELEGIBILIDADE SOC

**🚨 ARMADILHA #1: INCOMPATIBILIDADE EDUCACIONAL**
- **SE peticionário TEM bacharelado** → NUNCA usar códigos que marcam "No formal educational credential"
- **SE peticionário NÃO TEM bacharelado** → NUNCA usar códigos que exigem "Bachelor's degree"
- **Verificação obrigatória**: Cruzar formação do cliente com requisitos SOC no O*NET

**🚨 ARMADILHA #2: PROFISSÕES REGULAMENTADAS**
**CÓDIGOS PROIBIDOS** (exigem validação/licenciamento nos EUA):
- Médicos (29-1XXX) → **USE**: 11-9111 Medical and Health Services Managers
- Engenheiros (17-2XXX) → **USE**: 11-9041 Architectural and Engineering Managers  
- Advogados (23-1011) → **USE**: 11-9199 Managers, All Other
- Contadores (13-2011) → **USE**: 11-3031 Financial Managers
- Arquitetos (17-1011) → **USE**: 11-9041 Architectural and Engineering Managers

**🚨 ARMADILHA #3: INCONSISTÊNCIA COM EVIDÊNCIAS**
- Código deve permitir usar MÁXIMO de evidências EB-1A disponíveis
- Verificar se atividades documentadas se alinham com descrição SOC
- Garantir que critérios comprovados sejam típicos da ocupação escolhida

#### METODOLOGIA DE SELEÇÃO BASEADA EM EVIDÊNCIAS

**PASSO 1: ANÁLISE DOS CRITÉRIOS DISPONÍVEIS**
Com base no arquivo "EB-1-Analise-Aprofundada", identifique quais dos 10 critérios o peticionário possui evidências sólidas:

1. Prêmios nacionais/internacionais
2. Associações de elite
3. Material publicado sobre o peticionário
4. Atuação como juiz/avaliador
5. Contribuições originais significativas
6. Autoria de artigos acadêmicos
7. Exposições/mostras artísticas
8. Papel de liderança crítica
9. Alta remuneração
10. Sucesso comercial artes cênicas

**PASSO 2: MATCHING ESTRATÉGICO SOC x EVIDÊNCIAS**
- Selecionar código onde evidências documentadas são COMUNS na ocupação
- Evitar códigos onde atividades do peticionário sejam atípicas
- Priorizar códigos que naturalmente justifiquem múltiplos critérios

**PASSO 3: VALIDAÇÃO CONTRA PADRÕES DE NEGAÇÃO USCIS**
Baseado na análise 2023-2025, verificar se código escolhido evita:
- Evidências genéricas sem substância
- Falta de impacto/alcance comprovado  
- Contribuições limitadas ao empregador
- Ausência de reconhecimento externo

### PROTOCOLO DE SEGURANÇA

**NUNCA:**
- Recomendar código sem verificar requisitos educacionais no O*NET
- Sugerir códigos para profissões regulamentadas
- Fazer análise genérica sem documentos do peticionário
- Ignorar incompatibilidades educacionais

**SEMPRE:**
- Cruzar formação acadêmica com requisitos SOC
- Usar códigos alternativos para profissões regulamentadas
- Maximizar uso das evidências disponíveis
- Justificar escolhas com base nos padrões USCIS

**RESULTADO FINAL:** Código SOC estratégico + análise de compatibilidade + maximização de evidências + evitação de armadilhas USCIS.
