### GATE 3.16 — Integridade XML Pós-Transformação

> **Lição:** Um script de formatação (bold) reverteu a ordem dos `<w:r>` runs dentro de parágrafos, destruindo a legibilidade de centenas de parágrafos. 4 versões (v19→v22) foram necessárias para corrigir.

**ANTES de qualquer script de transformação XML:**
- [ ] Backup completo do document.xml criado automaticamente
- [ ] Texto puro extraído (concatenação de todos `<w:t>`) e salvo como `pre_transform.txt`

**APÓS qualquer script de transformação XML:**
- [ ] Texto puro extraído novamente e salvo como `post_transform.txt`
- [ ] Comparação `diff pre_transform.txt post_transform.txt`:
  - Se QUALQUER parágrafo mudou de conteúdo textual → **FAIL IMEDIATO — reverter backup**
  - Se parágrafos mudaram de ordem → **FAIL IMEDIATO — reverter backup**
  - Apenas mudanças em `<w:rPr>` (formatação) são permitidas
- [ ] Hash MD5 dos primeiros 100 parágrafos antes/depois idêntico

**Regra absoluta:** Scripts de formatação NUNCA devem mover, reordenar, inserir ou remover elementos `<w:r>`. Apenas modificar `<w:rPr>` dentro de runs existentes.

**Script de verificação:**
```python
def verificar_integridade_textual(xml_antes, xml_depois):
    """Compara texto extraído antes e depois de transformação."""
    def extrair_paragrafos(xml_path):
        tree = ET.parse(xml_path)
        W = '{http://schemas.openxmlformats.org/wordprocessingml/2006/main}'
        body = tree.getroot().find(f'{W}body')
        result = []
        for p in body.findall(f'{W}p'):
            texts = [t.text for t in p.iter(f'{W}t') if t.text]
            result.append(''.join(texts))
        return result

    antes = extrair_paragrafos(xml_antes)
    depois = extrair_paragrafos(xml_depois)

    if len(antes) != len(depois):
        return False, f"Contagem de parágrafos mudou: {len(antes)} → {len(depois)}"

    for i, (a, d) in enumerate(zip(antes, depois)):
        if a != d:
            return False, f"Parágrafo {i} alterado: '{a[:50]}...' → '{d[:50]}...'"

    return True, "Integridade textual preservada"
```
