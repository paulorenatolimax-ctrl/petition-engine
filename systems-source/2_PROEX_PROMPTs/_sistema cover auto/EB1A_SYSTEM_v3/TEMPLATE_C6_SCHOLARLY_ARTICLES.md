## BLOCO 1: LEGAL FRAMEWORK

### Parágrafo de abertura
```
O Critério 6 do EB-1A exige evidência da autoria de artigos científicos na área de 
atuação do peticionário, publicados em periódicos profissionais ou de grande circulação 
ou outros grandes meios de comunicação. [Citar 8 C.F.R. § 204.5(h)(3)(vi)]
```

### PA-2025-16 (usar defensivamente)
```
Importa destacar um desenvolvimento regulatório decisivo: a PA-2025-16 transformou 
a adjudicação EB-1A de discricionária para não-discricionária, eliminando a exigência 
anterior de "demonstrar o valor" das publicações.
```

### Definição ampla de "scholarly articles"
```
A USCIS Policy Manual e precedentes da AAO reconhecem que a avaliação de "scholarly 
articles" não se limita a artigos em formato journal — o termo abrange "professional 
publications or other major media," incluindo publicações técnicas, manuais, relatórios 
e livros que contribuem para o corpo de conhecimento do campo.
```

## BLOCO 2: PUBLICAÇÃO PEER-REVIEWED

### Evidence Block + Subseções:

#### I. Artigo Publicado em Periódico Acadêmico Peer-Reviewed
- Título completo, periódico, volume, número, data
- Processo de peer review descrito
- Metodologia (resumo breve)
- Achados/resultados principais

#### Nota sobre o Periódico (DEFESA PREEMPTIVA)
Se o periódico não é de alto impacto internacional:
```
[Nome do periódico] é periódico acadêmico mantido por [instituição]. 
[Instituição] é [credenciais: MEC, número de alunos, reconhecimento].
O processo editorial inclui [peer review, comitê editorial, etc.].
```
Defesa: PA-2025-16 removeu exigência de demonstrar "valor" ou "fator de impacto"

## BLOCO 3: DISSERTAÇÃO/TESE (se houver)

#### II. Credencial Acadêmica e Vínculo Dissertação → Publicação
- Programa, universidade, nota/classificação
- Créditos ECTS, disciplinas relevantes (mostrar rigor)
- Vínculo direto: dissertação → artigo publicado
- Pipeline: pesquisa → dissertação → publicação → aplicação comercial

```
A trajetória pesquisa → dissertação de mestrado ([nota]/10) → publicação em 
periódico peer-reviewed → aplicação comercial ([produto]) demonstra pipeline 
completo de transferência de conhecimento.
```

---

## BLOCO 4: COBERTURA INSTITUCIONAL (se houver)

#### III. Divulgação Científica da Publicação
Se o periódico/universidade publicou notícia sobre o artigo:
```
O significado probatório desta cobertura institucional é triplo:
1. Reconhecimento editorial duplo (periódico + portal de notícias)
2. Curadoria narrativa (transformação do formato acadêmico em narrativa acessível)
3. Disseminação amplificada (alcance institucional somado ao alcance do periódico)
```

---

## BLOCO 5: RELATÓRIO TÉCNICO-CIENTÍFICO (se houver)

#### IV. Documentação Científica de Metodologia Proprietária
- Descrição do documento e seu escopo
- Mecanismos científicos documentados
- Bibliografia com journals de alto impacto (NEJM, Cell, JAMA, etc.)
- Dados de coorte (tamanho amostral, resultados)

#### Nota sobre o Formato (DEFESA PREEMPTIVA)
```
Embora não publicado em formato de artigo peer-reviewed, o relatório constitui 
contribuição intelectual substancial ao campo. A USCIS Policy Manual reconhece 
que "scholarly articles" abrange "professional publications" — o foco é na 
substância da contribuição intelectual, não no formato de publicação.
```

## BLOCO 7: LIVROS PUBLICADOS (se houver)

#### VI. Obras Publicadas
- Títulos, ano, autoria exclusiva
- Consistência temática com o campo
- Precedentes AAO para livros como "scholarly articles"

**CUIDADO**: Se os livros não têm ISBN/editora documentada, NÃO mencionar ISBN. NÃO inventar editora. Argumentar pela substância do conteúdo, não pelo formato editorial.

```
A USCIS e a AAO reconhecem que livros técnicos publicados constituem "scholarly 
articles" quando o conteúdo é substancial e contribui para o corpo de conhecimento.
```

## VULNERABILIDADES CONHECIDAS E DEFESAS

| Vulnerabilidade | Defesa |
|----------------|--------|
| Apenas 1 artigo peer-reviewed | PA-2025-16 foco qualidade>quantidade + complemento com outros formatos |
| Livros possivelmente self-published | NÃO mencionar ISBN. Argumentar pela substância |
| Relatório técnico proprietário | Policy Manual aceita "professional publications" |
| Manual não é publicação formal | Documento com PI registrada + dados de coorte = substancial |
| Periódico de baixo impacto | MEC credenciado + peer review + tamanho da instituição |
| Dissertação não é artigo | Vínculo dissertação→publicação demonstra pipeline acadêmico |
