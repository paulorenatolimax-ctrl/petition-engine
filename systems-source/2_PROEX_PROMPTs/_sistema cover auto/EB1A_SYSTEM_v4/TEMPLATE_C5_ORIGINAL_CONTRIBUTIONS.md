## BLOCO 3: CADA CONTRIBUIÇÃO

### Estrutura por contribuição:

#### I. Natureza e Originalidade
- O que é a contribuição
- Como é original (primeira no campo, abordagem inédita, etc.)
- Propriedade intelectual associada (marcas INPI, patentes, ISBN)
- Evidence links

#### II. Fundamentação (Científica/Técnica/Empresarial)
- Manual técnico, publicação, protocolo documentado
- Referências a literatura (journals de alto impacto)
- Dados que sustentam a fundamentação
- Evidence links

#### III. Major Significance (Escala e Impacto)
- Métricas de adoção (usuários, clientes, participantes)
- Métricas de alcance (seguidores, views, downloads)
- Reconhecimento por plataformas (Top Producer, certificações)
- Dados financeiros de monetização
- Evidence links

#### IV. Validação por Terceiros
Para CADA contribuição, idealmente 2+ validadores:
```
[Nome do Validador] — [Credenciais brevíssimas: título, seguidores/empresa, especialidade]

Na carta de recomendação (Evidence XX), [Nome] documenta/atesta que [impacto específico].
[Detalhamento do que a carta diz — dados concretos, transformação documentada]
[Citação parcial relevante entre aspas se disponível]
```

#### V. Replicabilidade Cross-Industry (se aplicável)
Quando a mesma contribuição foi aplicada em setores diferentes:
```
A convergência de validações — por [Validador A] no campo [X] e por [Validador B] 
no campo [Y] — demonstra que a contribuição possui aplicabilidade transversal, 
transcendendo [setor original] para impactar [outros setores].
```

## PADRÕES ARGUMENTATIVOS EFICAZES

### Validação Cruzada
2+ recomendadores de setores diferentes validando a MESMA contribuição:
```
"A convergência de validações por terceiros — pelo [Validador A] no campo [X] 
e pelo [Validador B] no campo [Y] — demonstra que a contribuição não é 
circunscrita a um único setor, mas possui impacto transversal."
```

```
"A escala de adoção — [N] participantes/usuários/clientes — posiciona esta 
contribuição entre as [maiores/mais amplamente adotadas] no campo de [campo]."
```

```
"A trajetória [pesquisa] → [publicação peer-reviewed] → [aplicação comercial] 
→ [registro PI] → [receita documentada] demonstra pipeline completo de 
transferência de conhecimento."
```
