# TEMPLATE RÉSUMÉ EB-2 NIW — Estrutura por Seção (V2.0 — Layout Premium)

> Este template define a estrutura EXATA de cada seção do résumé EB-2 NIW.
> O layout visual (cores, fontes, evidence blocks) é IDÊNTICO ao EB-1A.
> A ESTRUTURA DE CONTEÚDO é adaptada para o framework EB-2 NIW (Dhanasar).

---

## 1. HEADER — Barra Navy Premium

Idêntico ao EB-1A, exceto:
```
Row 1: RÉSUMÉ          SOC/O*Net: XX-XXXX.XX | EB-2 NIW
```
(EB-2 NIW em vez de EB-1A)

## 2. SÍNTESE PROFISSIONAL / EXECUTIVE SUMMARY (1-2 páginas)

### Navy Section Header:
```
┌─── NAVY BAR ─────────────────────────────────────────────┐
│  SÍNTESE PROFISSIONAL  ou  EXECUTIVE SUMMARY             │
└──────────────────────────────────────────────────────────┘
```

### Estrutura Interna (4-8 parágrafos, Garamond 10.5pt, justificados):

**Parágrafo 1 — Quem é**
Credenciais, registro profissional, anos de experiência. Posicionamento no campo.

**Parágrafo 2 — O que criou/desenvolveu**
Contribuição original principal. Metodologia, produto, sistema ou inovação central.

**Parágrafo 3 — Impacto quantificado**
Números: projetos, clientes, receita, países, downloads, etc.

**Parágrafo 4 — Por que é qualificado (Advanced Degree + Exceptional Ability)**
Demonstrar grau avançado OU habilidade excepcional equivalente.

**Parágrafo 5 — Proposed Endeavors (preview)**
Breve menção das propostas de atuação nos EUA e seu alinhamento com interesse nacional.

**Parágrafos 6-8 (opcionais)**
Reconhecimentos, expansão internacional, atuação em setores específicos.

### REGRA DE PROFUNDIDADE
Cada parágrafo: **MÍNIMO 4-6 linhas**. PROIBIDO parágrafos mirrados.

### Idioma
- Padrão: **Inglês**
- Se Paulo instruir: **Português**

## 10. PROJETO EB-2 NIW — PROPOSED ENDEAVORS ⭐ (SEÇÃO PRINCIPAL)

**Esta é a seção que diferencia o EB-2 NIW do EB-1A.**

### Navy Section Header:
```
┌─── NAVY BAR ─────────────────────────────────────────────┐
│  PROJETO EB-2 NIW — PROPOSED ENDEAVORS                    │
└──────────────────────────────────────────────────────────┘
```

### 10.1 Parágrafo Introdutório
Contextualizar o framework EB-2 NIW (Dhanasar, 2016):
- Prong 1: Substantial merit and national importance
- Prong 2: Well positioned to advance the proposed endeavor
- Prong 3: Balance of equities — benefit to the United States

### 10.2 Para cada Proposed Endeavor:

#### Teal Sub-Header:
```
┌─── TEAL BAR ─────────────────────────────────────────────┐
│  Proposta A: [Título da Proposta]                         │
└──────────────────────────────────────────────────────────┘
```

#### Conteúdo da Proposta (4-6 parágrafos):

**Parágrafo 1 — Descrição da proposta**
O que o beneficiário pretende fazer nos EUA. Qual produto/serviço/empresa.

**Parágrafo 2 — Dados de mercado**
TAM/SAM/SOM, projeções de crescimento, BLS codes, fontes (Gartner, Statista, etc.)

**Parágrafo 3 — Alinhamento com políticas governamentais**
Executive Orders, CHIPS Act, NIST frameworks, DoL priority areas, etc.

**Parágrafo 4 — Por que o beneficiário é well-positioned**
Track record, certificações, projetos similares, contribuições open-source.

**Bullets com dados quantitativos:**
```
● Market size: $XX billion by 2030 (Source: Gartner)
● BLS Occupational Code: XX-XXXX — projected XX% growth 2024-2034
● Federal alignment: [nome do programa/política]
```

### 10.3 Tabela Comparativa das Propostas

```

│ Dimensão        │ Proposta A  │ Proposta B  │ Proposta C │

│ Mercado-alvo    │ ...         │ ...         │ ...        │
│ BLS Code        │ ...         │ ...         │ ...        │
│ Crescimento     │ ...         │ ...         │ ...        │
│ Política federal│ ...         │ ...         │ ...        │
│ Receita projetada│...         │ ...         │ ...        │
└─────────────────┴─────────────┴─────────────┴────────────┘
```

- Header: Navy bg, White text
- Alternate rows: #F5F5F5 / #FFFFFF
- Garamond 9.5pt

### 10.4 Parágrafo Conclusivo (Dhanasar)

Parágrafo final ligando as 3 propostas ao framework legal:
- Como cada proposta tem "substantial merit and national importance"
- Como o beneficiário é "well positioned"
- Como aprovar o caso beneficia os EUA mais do que negar

## 11. SEÇÕES QUE NÃO EXISTEM NO EB-2 NIW

O résumé EB-2 NIW NÃO tem:
- ❌ Seções por critério (C1-C10 são do EB-1A)
- ❌ Awards/Prêmios como seção separada (podem aparecer em Contribuições)
- ❌ Membership como seção separada
- ❌ High Salary como seção separada
- ❌ Commercial Success
